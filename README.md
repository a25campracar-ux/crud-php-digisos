# Desenvolupament d’un CRUD en PHP (MySQLi) en servidor real

En aquest projecte he creat una aplicació web connectada a una base de dades MySQL que permet gestionar un registre de videojocs. L’aplicació disposa d’una interfície senzilla per realitzar operacions bàsiques de CRUD (crear, llegir, actualitzar i eliminar registres), implementades amb PHP i MySQLi. A més, el projecte està desplegat en un servidor real, cosa que permet accedir-hi des de qualsevol dispositiu amb connexió a internet i comprovar el funcionament complet de l’aplicació en un entorn més proper a la realitat.
